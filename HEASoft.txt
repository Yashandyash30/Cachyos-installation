Installing HEASoft (or Xspec)

    Install HEASoft into a new conda environment (called henv):

    mamba create -n henv heasoft \
      -c https://heasarc.gsfc.nasa.gov/FTP/software/conda/ \
      -c conda-forge

Activate the newly created conda environment:

conda activate henv

The prompt should change to something like (henv)[...]. HEASoft should be initialized automatically.
Start using HEASoft.

If you plan to use Xspec, make sure you add the model files available in the xspec-data package when installing HEASoft: 
Since your environment is already built and activated (as shown by the (henv) in your prompt), you don't need to create it again. You just need to add the missing data package to it.
Run this command instead:

mamba install xspec-data -c https://heasarc.gsfc.nasa.gov/FTP/software/conda/ -c conda-forge

Mamba will safely download just the 5GB of XSPEC atomic data files and slot them directly into your existing setup without wiping anything out.
